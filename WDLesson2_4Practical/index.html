Complex Selectors

The element, class and id selectors can be combined to create more complex selectors. Notice how the above heading and this paragraph are unaffected by the CSS rules in Challenges 4 and 5

.container h1

The above selector demonstrates how to select all the headings in elements with a class of "container"

.container p

The above selector demonstrates how to select all the paragraphs in elements with a class of "container"

#testArea h1

The above selector demonstrates how to select all the headings within an element with an id of testArea

Two of more selectors can be specified for a CSS rule by separating the selectors with comma.
a, div, span{
  text-shadow:2px 2px red;
}
 Above CSS rule would add a shadow to the text for all anchors,containers both div and span
